## Dockerコマンドの確認方法
```bash
docker --help
```

### Dockerのバージョン確認（インストールされているかの確認にも使える）
<hr>

```bash
docker version
```

###　Dockerの詳細情報確認
<hr>

```bash
docker info
```

### コマンドの確認
<hr>

```bash
docker --help 
    or 
docker
```

### Docker コマンドの基本構造
<hr>

```bash
new: docker <sub-command> (options) 
    or
old: docker <command> (options)
```
